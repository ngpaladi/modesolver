# GitHub Pages Configuration

This repository is configured to host documentation via GitHub Pages.

## Build Process

Documentation is built using Sphinx and automatically deployed to GitHub Pages.

## Directory Structure

- `docs/` - Sphinx source files
- `docs/_build/` - Generated HTML documentation (auto-generated)
- `docs/index.html` - Main entry point for GitHub Pages

## Deployment

The documentation is automatically deployed via GitHub Actions when changes are pushed to the main branch.
